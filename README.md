\# 20Newsgroups 二分类任务（单层GRU）



\## 项目简介

使用 `20newsgroups` 数据集中的两个类别：

\- `alt.atheism`（无神论）

\- `soc.religion.christian`（基督教）



构建一个\*\*单层 GRU\*\* 模型完成二分类任务，测试准确率高于 0.7。



\## 数据预处理

\- 转换为小写

\- 移除 HTML 标签

\- 移除标点符号

\- 移除数字

\- 去除多余空格

\- 词汇表：保留出现次数 ≥2 的词，包含 `<PAD>`（索引0）和 `<UNK>`（索引1）



\## 数据划分

\- 原始训练集：1079 条

\- 原始测试集：717 条

\- 从训练集中分出 20% 作为验证集（分层抽样）

\- 最终：训练集 863 条，验证集 216 条，测试集 717 条



\## 模型结构（单层 GRU）

| 组件          | 参数                        |

|---------------|-----------------------------|

| 嵌入层        | 维度 256，padding\_idx=0      |

| GRU 层        | 输入 256，隐藏 256，层数 1   |

| Dropout       | 丢弃率 0.5（位于全连接层前） |

| 全连接层      | 输入 256，输出 1             |

| 损失函数      | BCEWithLogitsLoss            |



总参数量：约 3.2M



\## 超参数选择

| 超参数          | 取值   | 选择理由                                 |

|----------------|--------|------------------------------------------|

| 嵌入维度        | 256    | 增强表示能力                             |

| 隐藏层维度      | 256    | 与嵌入维度一致，捕获序列依赖             |

| Dropout         | 0.5    | 防止过拟合                               |

| 批量大小        | 64     | 典型值，稳定梯度                         |

| 学习率          | 0.001  | Adam 默认值，收敛稳定                    |

| 权重衰减        | 1e-4   | 轻微 L2 正则化                           |

| 最大序列长度    | 60     | 覆盖主要信息，避免梯度消失               |

| 早停            | 耐心 6 | 防止过拟合，节省训练时间                  |



\## 优化方法

\- \*\*优化器\*\*：Adam（自适应学习率）

\- \*\*学习率调度\*\*：无（固定学习率，早停控制）
